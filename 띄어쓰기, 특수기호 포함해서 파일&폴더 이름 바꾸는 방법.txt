[[[[[[[[파일 및 폴더 이름 바꾸기]]]]]]]]]]]]
(단, 1.이름 바꿀 대상 폴더가 비어있거나, 
2.git add 명령으로 stage에 올라가있지 않은 경우,
폴더를 채우거나 add명령 해주면 이름 바꿀 수 있다.)

' " 를 사용하면 이름에 띄어쓰기, 특수기호 거의 다 사용 가능
(cd로 접근 할 때도 이 기호를 쓰면 특수기호나 띄어쓰기가 포함 된 경로도 접근 가능)

------------모두 가능----------------
바꿀 폴더 명 : 001
$ git mv 001 0' '' '1
$ git mv 001 '001 & @ _002'
$ git mv 001 001' ''&'' ''@ '_003
$ git mv 001 "001 '&' @ _004"
$ git mv 001 001" & @ _"004
------------------------------------

도 마찬가지.
